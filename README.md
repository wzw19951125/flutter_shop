一个基于flutter的线上商城项目，后端使用的是it营提供的接口

目前实现：
商城首页展示
商品分类选择

目前使用的插件：
flutter_swiper_null_safety 轮播图
cached_network_image 带缓存的网络图片
dio 网络请求
